# PDF Encryption Script

A simple Python utility to encrypt a PDF file using a password.  
Built using the `PyPDF2` library.

---

## Features
- Encrypts a PDF file with a user-provided password
- Preserves all original pages
- Prevents access to the PDF without the correct password
- Basic error handling for common issues

---

## Requirements
- Python 3.x
- PyPDF2 library

Install PyPDF2 using pip:
```bash
pip install PyPDF2
